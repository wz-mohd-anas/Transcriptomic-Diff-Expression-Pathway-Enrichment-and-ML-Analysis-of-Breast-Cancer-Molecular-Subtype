# Transcriptomic-Diff-Expression-Pathway-Enrichment-and-ML-Analysis-of-Breast-Cancer-Molecular-Subtype
Integrated transcriptomic and machine learning analysis of breast cancer molecular subtypes using Orange Data Mining, GEO2R, Enrichr, and STRING. The project includes PCA, clustering, differential expression analysis, pathway enrichment, biomarker discovery, and subtype classification.
